Sub Bridge

Use ChatGPT Pro/Max, Claude Max, etc. in Cursor via MCP.

Quick start (Cursor MCP):

  • Add an MCP server with npx and args -y sub-bridge.
  • Call the tool get_status in Cursor chat.
  • Set your Base URL to <publicUrl>/v1 and API Key to sk-ant-xxx,sk-xxx.

Tokens: run claude setup-token or codex login to get a token.
